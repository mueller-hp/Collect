# מערכת ניהול גבייה מתקדמת | Advanced Collections Management System

> אפליקציית WEB client-side לניתוח חובות, התראות אוטומטיות, ודוחות מפורטים

## 🎯 תיאור המערכת

מערכת ניהול גבייה מתקדמת המאפשרת קריאה וניתוח של קבצי Excel עם נתוני חובות לקוחות. המערכת פועלת לחלוטין בדפדפן (client-side) ללא צורך בשרת, ומותאמת לפרסום ב-GitHub Pages.

## ✨ תכונות עיקריות

- 📊 **ניתוח חובות מתקדם** - זיהוי מקרים קריטיים ורמות סיכון
- 🚨 **התראות אוטומטיות** - התראות ויזואליות ובדפדפן
- 📈 **גרפים ותרשימים** - ויזואליזציה של נתוני הגבייה
- 📋 **טבלאות אינטראקטיביות** - מיון, חיפוש ויצוא נתונים
- 🌐 **תמיכה בעברית RTL** - ממשק מלא בעברית
- 📱 **עיצוב רספונסיבי** - תואם לכל המכשירים
- 💾 **שמירה לוקלית** - כל הנתונים נשמרים במכשיר שלך

## 🚀 התחלה מהירה

### אפשרות 1: GitHub Pages (מומלץ)
1. עבור ל-[https://gil100.github.io/Collections/](https://gil100.github.io/Collections/)
2. טען קובץ Excel עם נתוני החובות
3. התחל לנתח!

### אפשרות 2: הפעלה מקומית
1. הורד את הקבצים או שכפל את הרפוזיטורי
2. פתח את `index.html` בדפדפן
3. טען קובץ Excel והתחל לעבוד

## 📋 דרישות קובץ Excel

הקובץ צריך לכלול את העמודות הבאות:
- **שם חייב** - שם הלקוח או החברה
- **מספר זהות** - מספר תעודת זהות/ח.פ.
- **טלפון** - מספר טלפון ליצירת קשר
- **כתובת** - כתובת הלקוח
- **סכום חוב** - הסכום הכולל בש"ח
- **תאריך חוב אחרון** - תאריך החוב האחרון
- **ימים מאז תשלום** - מספר הימים מאז התשלום האחרון
- **קטגוריית סיכון** - רמת הסיכון (או יחושב אוטומטית)
- **הערות** - הערות נוספות

## 🛠️ טכנולוגיות

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **UI Framework**: Bootstrap 5
- **Charts**: Chart.js
- **Tables**: DataTables
- **Excel Reading**: SheetJS
- **Icons**: Font Awesome

## 🔧 מודולי המערכת

המערכת כוללת 6 מודולי JavaScript עיקריים:

1. **debt-engine.js** (576 שורות) - מנוע ניתוח החובות המרכזי
2. **dashboard.js** (615 שורות) - ניהול הדשבורד וכרטיסי KPI
3. **file-handler.js** (523 שורות) - טיפול בהעלאת קבצים וגרירה
4. **chart-config.js** (661 שורות) - תצורות גרפים מתקדמות
5. **datatables-config.js** (407 שורות) - הגדרות טבלאות אינטראקטיביות  
6. **alert-system.js** (478 שורות) - מערכת התראות מתקדמת

## 🔧 פיתוח

המערכת בנויה כאפליקציית client-side טהורה ללא תלויות שרת.

```bash
# הפעלת שרת פיתוח פשוט (אופציונלי)
python -m http.server 8000
# או עם Node.js
npx http-server
```
