# קבצי דוגמה | Example Files

תיקייה זו מכילה קבצי דוגמה לבדיקת המערכת.

## קבצים זמינים:

### sample-debts.csv
קובץ CSV לדוגמה עם נתוני חובות לקוחות מדומים. כולל:
- 15 לקוחות לדוגמה
- נתונים מגוונים של חובות וימי פיגור
- דוגמאות למקרים קריטיים
- חייבים ישראלים וזרים

### template.csv
תבנית ריקה לקובץ חובות עם:
- כותרות מוכנות בעברית
- מבנה נכון לייבוא
- שדות נדרשים

## המרה ל-Excel:
קבצי ה-CSV ניתנים להמרה בקלות ל-Excel:
1. פתח Excel
2. לחץ על "קובץ" → "פתח"
3. בחר קובץ CSV
4. שמור כ-XLSX

## עמודות נדרשות:
- שם חייב
- מספר זהות
- טלפון  
- כתובת
- סכום חוב
- תאריך חוב אחרון
- ימים מאז תשלום
- קטגוריית סיכון
- הערות

## שימוש:
1. הורד את הקובץ המתאים
2. המר ל-Excel (אם נדרש)
3. טען אותו במערכת
4. התחל לנתח את הנתונים

**הערה**: קבצים אלו מיועדים לבדיקה ולהדגמה בלבד. הנתונים פיקטיביים.
